# 推特发布计划与节奏

---

## 一、Thread 发布的核心规则

### ✅ 必须做
- **一次性连发**（用推特"+"按钮添加多条，最后一次性发布）
- **每条之间间隔为0**（同时发出去）
- 发完立即设置为**置顶**

### ❌ 绝对不能做
- ❌ 拆开几小时甚至几天发（会被算法当成独立推文，丧失 thread 加权）
- ❌ 在第1条下面手动回复式发后续（间隔太长会断 thread 链）
- ❌ 加超过1个 hashtag（推特对多 tag 是降权信号）
- ❌ 推文里放外链（降权 20-40%，要放就放回复里或图片中）

---

## 二、最佳发布时间（北京时间）

| 时段 | 活跃度 | 适合发什么 |
|---|---|---|
| 早 8-10点 | ⭐⭐ | 不推荐发重要内容 |
| 中 12-14点 | ⭐⭐⭐ | 短推文可以 |
| 下午 18-20点 | ⭐⭐⭐⭐ | 很好 |
| **晚 21-24点** | ⭐⭐⭐⭐⭐ | **置顶 thread 最佳时段** |
| **凌晨 0-3点** | ⭐⭐⭐⭐⭐ | **盘子时段，链上日记最佳** |
| 凌晨 3-6点 | ⭐⭐ | 美区开始活跃 |

**置顶 thread 的最佳发布时间**：
- 工作日 22:00-23:00
- 不要选周末（用户不刷推特）
- 不要选凌晨（虽然活跃但都在看盘，没耐心读长文）

## 三、发布前 30 分钟准备清单

发 Thread 前 30 分钟必须完成：

- [ ] 把每一条内容存到备忘录（防止发的过程中卡顿丢失）
- [ ] 检查每条字数 < 280 字符
- [ ] 准备好需要配的图（账户截图打码）
- [ ] 列出 5 个要去评论的中文 CT 大V账号
- [ ] 列出能发链接的群（如有）
- [ ] 关闭其他干扰，专注 1 小时

---

## 四、发布操作流程（标准动作）

### Step 1：22:00 整点 — 一次性连发
1. 打开推特发推界面
2. 输入第1条
3. 点 "+" 添加第2条
4. 重复直到最后一条
5. **一次性点"全部发布"**

### Step 2：22:01-22:05 — 立即激活
1. 给自己第1条**点赞**
2. 在第1条下方回复一句**互动引导**：
   ```
   忘了说一句：评论区欢迎晒你的捡垃圾持仓，互相对账 👇
   ```
3. 把链接发到任何能发的群（如有）

### Step 3：22:05-22:30 — 蹭流量
1. 去 5 个中文 CT 大V最新帖子下**有信息量地评论**
2. 评论里**自然带出你的观点**，不是 +1 这种水评论
3. 让看到的人对你产生兴趣 → 点你头像 → 看到 Thread → 关注

### Step 4：22:30-23:30 — 持续互动
1. 任何来你 thread 评论的，**立即回复**
2. 持续刷推特，看到同赛道帖子就有质量地评论

### Step 5：23:30 — 收尾
1. 把这条 thread **设为置顶**
2. 截图当晚的展示数据存档（用于后续复盘）
3. 收工

## 七、避坑清单

### 推特算法红线
- ❌ 短时间发太多重复内容（被识别为机器人）
- ❌ 频繁 @ 大V且内容无关（被举报会扣权重）
- ❌ 一条推文超过 3 个 hashtag
- ❌ 推文带 t.me / 收款码图片（疑似营销/诈骗判定）

### 内容红线
- ❌ 喊单（"建议买入XXX"）
- ❌ 带单（"跟我下单"）
- ❌ 收费暗示（"加群XXX")
- ❌ 承诺收益（"必涨""稳赚"）

**安全表达**：永远是"我做了什么"，不是"你应该做什么"。
