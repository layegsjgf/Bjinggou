# 平台风控规则与避坑

---

## 一、推特(X) 风控

### 风险等级：极低

### 注意事项

| 行为 | 影响 |
|---|---|
| 加 1 个 hashtag | 无影响 |
| 加 3+ hashtag | 降权 20-40% |
| 推文带外链 | 降权 20-40% |
| 短时间发 10+ 条相似内容 | 触发机器人判定 |
| 频繁 @ 不相关大V | 被举报后扣权重 |

### 推特上可以直接说的词

- ✅ Solana / SOL / Sol meme
- ✅ 币圈 / 链上 / 钱包 / 交易所名
- ✅ 买入 / 卖出 / 盈利 / 亏损
- ✅ 喊单 / 带单（指责别人时可用）

**结论：推特就是币圈母语区，正常说话即可。**

## 二、公众号风控

### 风险等级：中等

### 绝对红线词（出现就删文/警告）

```
比特币 / 以太坊 / 莱特币 / 瑞波币 / 狗狗币
BTC / ETH / SOL / DOGE / BNB
币安 / OKX / 火币 / Coinbase / Bybit
虚拟货币 / 数字货币 / 加密货币 / Crypto
炒币 / 买币 / 卖币 / 持币 / 囤币
挖矿 / 矿机 / 矿池
ICO / IEO / 空投 / Airdrop
合约交易 / 杠杆 / 爆仓 / 清算
代币 / Token / Coin
割韭菜
```

### 安全替换对照表

| 危险词 | 公众号安全写法 |
|---|---|
| Solana | Sol生态 / S链 / 某公链 |
| SOL（币） | 链上原生资产 / gas |
| meme币 | 链上微盘标的 / 社区驱动型项目 |
| 买入/卖出 | 建仓/清仓 / 开仓/平仓 |
| 赚了多少钱 | 正收益 X% |
| 亏了多少钱 | 回撤 X% / 负收益 |
| 交易 | 链上操作 / 策略验证 |
| 交易所 | 不提，只说"链上" |
| 钱包地址 | 链上地址（最好不放完整地址） |
| 炒币 | 链上数据研究 / 链上策略实验 |
| 韭菜 | 散户 / 普通参与者 |
| 币圈 | 链上生态 |
| 聪明钱 | 高胜率地址 / 标杆地址 |
| 庄家 | 主力资金 / 主控地址 |
| 跟庄 | 跟踪主力动向 |
| 梭哈 | 重仓 / 满仓 |
| 翻倍/10x | 波动空间X倍 / 涨幅X00% |
| 爆仓 | 触发强制止损 / 超出风控阈值 |

### 公众号定位话术

❌ 「币圈博主」
✅ 「链上数据研究笔记」
✅ 「交易系统构建日志」
✅ 「链上策略实验室」

### 公众号安全发布清单

发布前必检查：
- [ ] 标题没有任何敏感词
- [ ] 正文没有任何敏感词（用对照表替换）
- [ ] 截图打码：币种名/钱包地址/具体金额
- [ ] 没有放交易所链接、邀请码、合约地址
- [ ] 没有"推荐你买/关注 XXX"这种话术
- [ ] 文末加免责声明：
  ```
  ⚠️ 免责声明：
  本文仅为个人学习研究笔记，记录链上数据观察与策略思考过程。
  不构成任何投资建议，不推荐任何具体标的。
  链上操作风险极高，请读者独立判断、自担风险。
  ```
- [ ] 没开赞赏功能
- [ ] 公众号名字里不带"币""Crypto""Web3"

## 三、小红书风控

### 风险等级：极高

### 当前策略：暂不做

**原因**：小红书 2023 年后明确封杀虚拟货币内容，限流极严。

### 未来如果做的话（绕道方案）

不能提任何币圈关键词，**改做两个方向**：
1. **AI 工具教程号**："用 AI 自动写公众号""Coze 搭建客服机器人"
2. **副业/自由职业者日常**："33岁自由职业者的一天""副业第3个月收入公开"

引流到公众号 → 公众号再连接币圈内容。

## 四、内容上传前的"AI审核流程"

每篇公众号文章发布前：

1. **写完原稿**（怎么想怎么写）
2. **对照本文档敏感词清单**自查替换
3. **拿不准的发给 AI（Claude/GPT）做终审**
4. **AI 输出"安全版"全文**
5. **检查图片**（打码处理）
6. **加免责声明**
7. **发布**

### 给 AI 审核的 prompt 模板

```
请帮我审核以下公众号文章是否会触发风控：

【需要识别的风险】
1. 币圈金融敏感词（如：比特币、SOL、meme币、交易所名等）
2. 推荐/喊单类话术
3. 收益保证类表述

【请输出】
1. 标记所有风险词
2. 给出安全替换建议
3. 输出最终安全版全文

【原文】
{粘贴你的原文}
```
