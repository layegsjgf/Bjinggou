# 内容选题库 - 5类内容轮换模型

> 每天15-30分钟产出一条优质推文的素材库

---

## 一、5类内容比例

| 类别 | 占比 | 频率 |
|---|---|---|
| 链上日记 | 40% | 每天 |
| 亏损复盘 | 20% | 每周 1-2 次 |
| 策略迭代 | 15% | 每 1-2 周 |
| 工具科普 | 15% | 每周 1-2 次 |
| 生活碎片 | 10% | 随心 |

---

## 二、各类内容模板

### 第1类：链上日记（每天必发）

**填空模板**：
```
{日期} 链上日记

今日观察：
1. {币1}：{在多少市值看到的}，{进了/没进/为什么}
2. {币2}：{同上}
3. {币3}：{同上}

今日操作：
✅ 入场 X 笔
✅ 离场 X 笔
日盈亏：{+/-} ${金额}

一句话感受：{今天最大的体会}
```

**真实案例**（演示用）：
```
5/25 链上日记

今日观察：
1. nl：信号在 40k mcap 提示，进了 0.3sol
2. MASSIVE：群里有人喊了一嘴，手贱追高 200k 进
3. UTYA：昨天捡的，今天小拉了一下 +5%

今日操作：
✅ 入场 3 笔
✅ 离场 2 笔
日盈亏：+$11

一句话感受：群里喊的时候就是最不该进的时候。
第4次犯这个错了，下次直接拉黑群消息再做决策。
```

---

### 第2类：亏损复盘（每周 1-2 次）

**填空模板**：
```
本周最蠢操作复盘

币：{币名}
亏损：{金额} / {%}

❌ 为什么进：{当时的想法}
❌ 违反了哪条规则：{自己定的规则}
❌ 进场时的情绪：{是否飘了/FOMO/赌气}

🧠 复盘：
- {链上其实有什么信号你忽略了}
- {你违反规则的根本原因}

📌 修正：{下次怎么改/加什么新规则}
```

**真实案例**（演示用）：
```
本周最蠢操作复盘

币：MASSIVE
亏损：-$3.45 / -7.8%

❌ 为什么进：群里有人喊，FOMO了
❌ 违反规则：不追已经从底部涨>5x的盘子
❌ 进场情绪：连赚2笔后飘了

🧠 复盘：
- top10 持仓55%，本来就该警觉
- 喊单的人其实是 dev 关联钱包
- 我自己飘了，第一时间没看链上数据

📌 修正：从下周起任何盘子从底涨超5x不追
       入场前必看 top10 持仓比例
```

### 第3类：策略迭代（每 1-2 周）

**填空模板**：
```
🔧 {策略名}V{版本号} 更新

上版本数据（{时间段}）：
- 胜率：{%}
- 盈亏比：{比例}
- 最大连亏：{次数}

发现的问题：{核心问题描述}

V{新版本} 调整：
- 新增：{规则}
- 修改：{规则}
- 删除：{规则}

下{时间}跑数据看效果。
```

### 第4类：工具科普（每周 1-2 次）

**填空模板**：
```
{工具名} 隐藏技巧：{一句话总结要解决的问题}

3步操作：
1. {步骤1}
2. {步骤2}
3. {步骤3}

为什么有用：{解决什么问题}

📷 截图说明（推荐配图）
```

**选题灵感**：
- GMGN 怎么过滤 dev 没 burn 的盘子
- Photon 隐藏功能：按创建时间排新盘子
- Solscan 看一个币的 top holder 是不是同一伙人
- Bullx 限价单怎么挂
- Trojan bot 多账户配置
- Axiom 抢开盘按键党配置
- 怎么判断聪明钱包是不是老鼠仓
- TG bot 监控频道推荐

### 第5类：生活碎片（随心，每周 2-3 条）

**模板**：随手写，不要刻意

**选题灵感**：
- 凌晨看盘的状态
- 看着账户曲线的内心独白
- 币圈黑话的吐槽
- 跟非币圈人解释你在干嘛的尴尬
- 老韭菜踩过的坑（旧时代）

---

## 三、内容产出工作流

### 你的最小动作（每天 15 分钟）

1. **打开备忘录**，把今天看到的盘子、做的操作、心情用大白话写下来
2. **对照模板**填进去
3. **发到推特**

### 高阶动作（每周 1 小时）

1. **周末抽1小时**，把当周的链上日记汇总成1篇周复盘
2. **打磨成 thread**，发周五晚上
3. **同步发到公众号**（用风控翻译术处理）
